### How to use:

* First you clone the repo with: 
```
  git clone https://github.com/KauanLazzarin/CRUD-Nodejs-MongoDB.git
  
```
> Or with the GitHub CLI 
 
```
  gh repo clone KauanLazzarin/CRUD-Nodejs-MongoDB
  
```
* Then you run:

```
  npm install
```
